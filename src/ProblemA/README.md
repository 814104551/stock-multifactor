## 调整回测

在 **SingleFactor.py** 中需要变动的对象包含**begin(开始日期),end(结束日期)**，需要注意如果变更了开始日期，需要同步变更**init**中的**cons_date**，**ratio(基准指数)**。

待回测的因子在**factorName.csv**中，修改即可变更回测因子

回测期间不要关闭at窗口，回测结束后报告生成至**reports.csv**
